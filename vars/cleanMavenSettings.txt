<h1>cleanMavenSettings</h1>
<p>
Removes a maven settings file generated by this library's <strong>generateMavenSettings()</strong> function.
</p>
<br>
<h2>Arguments</h2>
None
<br>
<h2>Requirements</h2>
None
<br>
<h2>Example</h2>
<pre>
<code>
post {
    always {
        cleanMavenSettings()
    }
}
</code>
</pre>
